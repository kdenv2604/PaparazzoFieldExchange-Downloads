# PaparazzoFieldExchange Downloads

This repository contains public binary downloads for PaparazzoFieldExchange.

The source repository is not published here. Use the Releases page to download
installers, portable builds, user guides and checksums.

PaparazzoFieldExchange is distributed as a free-to-use application under the
localized EULA files included with each release. The application is not published
as open source as a whole. Open-source notices apply only to the listed third-party
or framework components.

PaparazzoFieldExchange is especially convenient for Android developers who need to
move APKs to test devices quickly. You can send Android and Android TV application
builds directly to a nearby device for installation and checking over local Wi-Fi or
hotspot mode, without cloud storage, messenger uploads or a USB cable.

## What to download

- Windows: `PaparazzoFieldExchange-Windows-x64-vX.Y.Z.msi`, `PaparazzoFieldExchange-Windows-arm64-vX.Y.Z.msi` or `PaparazzoFieldExchange-Windows-x86-vX.Y.Z.msi`
- Android: `PaparazzoFieldExchange-Android-vX.Y.Z.apk`
- Linux desktop: `PaparazzoFieldExchange-Linux-x64-vX.Y.Z.tar.gz` or ARM64
- Legacy Windows: `PaparazzoFieldExchange-Portable-Windows-*.zip`
- Legacy Linux: `PaparazzoFieldExchange-Portable-Linux-*.tar.gz`
- FreeBSD: `PaparazzoFieldExchange-Portable-FreeBSD-*.tar.gz`

The universal Windows setup EXE may be produced by the main project build, but it
is not uploaded to every public prerelease because it is much larger than the
per-architecture MSI installers.

Verify downloads with `SHA256SUMS.txt` when possible.
Read the included `EULA_*.md`, `THIRD_PARTY_NOTICES_*.md` and
`OPEN_SOURCE_LICENSES.md` files before redistribution.
